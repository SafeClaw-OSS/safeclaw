SafeClaw is a passkey-secured proxy that keeps your API credentials encrypted at rest and invisible to the agents that use them. Zero plaintext. Zero trust required.

⚙️ Get Started 📊 Dashboard
🔑

Passkey Encryption

Your vault is sealed with WebAuthn PRF — only your biometric can unlock it. No passwords, no master keys.

🔐

End-to-End Encrypted

Credentials are encrypted in your browser before transmission. The server never sees plaintext keys.

🤖

Agent-Invisible

Agents route through the proxy and get responses — but never see, store, or log your actual API keys.

🧊

Lock Anytime

One tap to lock the vault. Secrets are wiped from memory and the proxy returns safe placeholder responses.

🔄

Dual Passkey Backup

Register two passkeys on different devices. Lose one, unlock with the other — no lockout risk.

⚡

Transparent Proxy

SSE streaming, all HTTP methods, auth injection — your tools work exactly the same, just more secure.
